North American Breweries

Map displays the number of North Americacn breweries per 50km inner diameter hexagon. The map inset shows the breweries of Denver, Colorado in a 2km inner diameter hexagon. Map data from the POI Factory, embedded images: Top: www.mentalfloss.com, Bottom: www.thedrinknation.com, CRS: North American Albers Equal Area Conic

Map created by Kevin Hogue 2016 using QGIS point in polygon method.

  • See the map: North American Breweries
  • Data Source: POI Factory

UK Men's Basketball Hometowns

This map was built using CartoDB georeferencing, point in polygon, and map production. Map displays the hometowns of UK Men's Basketball players from 1903 through 2016, by state and county. Zoom in on the map to view county data.

  • See the map: UK Mens Basketball Hometowns
  • Data Source: The Kentucky Basketball Statistics Project
